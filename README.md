## How do I install these formulae?

`brew install atopile/tap/<formula>`

Or `brew tap atopile/tap` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "atopile/tap"
brew "<formula>"
```
